# flutter_firebase_codelab

## 使用

1. [iOS] 下載 `GoogleService-Info.plist`，開啟 Xcode，放入 `Runner/Runner`（注意是子 Runner） 資料夾下
2. [android] 下載 `google-services.json` 放入 `android/app` 資料夾下
